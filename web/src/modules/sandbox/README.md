# react

## Sandbox folder

Create all your widgets here.
```shell
+-- name-component
|   +-- app //for testing build of widget
|   |   +-- index.tsx
|   |   +-- index.html
|   +-- build //for testing build of widget
|   +-- components //internal level components if required
|   |   +-- NameComponent.tsx
|   +-- models
|   |   +-- IName.ts
+-- styles
|   +-- _name.scss
|   +-- main.scss
+-- NameComponent.tsx //main file of component calling its main styles and all internal components components
```
