# react

## Widgets folder

Create all your widgets here.
```shell
+-- usage-name-widget
|   +-- app //for testing build of widget
|   |   +-- index.tsx
|   |   +-- index.html
|   +-- build //for testing build of widget
|   +-- components //internal widgets level components
|   |   +-- NameComponent.tsx
|   +-- models
|   |   +-- IName.ts
+-- styles
|   +-- _name.scss
|   +-- main.scss
+-- UsageNameWidget.tsx //main file of widget calling its main styles and all components
```
