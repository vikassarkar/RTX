# react

## Pages folder

Create all your widgets here.
```shell
+-- usage-name-Pages
|   +-- app //for testing build of Page
|   |   +-- index.tsx
|   |   +-- index.html
|   +-- build //for testing build of page
|   +-- components //internal page level components
|   |   +-- NameComponent.tsx
|   +-- models
|   |   +-- IName.ts
+-- styles
|   +-- _name.scss
|   +-- main.scss
+-- UsageNamePage.tsx //main file of page calling its main styles and all components
```
